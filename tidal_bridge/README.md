# TIDAL Bridge (Skeleton)

This folder will contain the open-source TIDAL Bridge implementation for AudioArchitect.

Planned responsibilities:

- Authenticate with TIDAL using TIDAL Developer Platform.[web:19][web:15]
- Read user playlists and basic metadata (no full-track downloads).
- Export/import playlists using the neutral formats in `docs/playlist_format.md`.
- Follow the constraints defined in `docs/bridge_lockdown.md`.
